

## 环境设置

1. Python版本：
```bash
Python 3.10.12
```

2. 创建虚拟环境（可选但推荐）：
```bash
python -m venv venv_mcp_rag
source venv_mcp_rag/bin/activate  # 在 Windows 上: venv\Scripts\activate
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```
